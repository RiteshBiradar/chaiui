## Props

| **Prop**        | **Type**    | **Description**                                      | **Default**                                                  |
|-----------------|-------------|------------------------------------------------------|--------------------------------------------------------------|
| `title`         | `string`    | Title text shown at the top of the card             | `"Course Title"`                                             |
| `subtitle`      | `string`    | Subtitle text shown below the title                  | `"Course Subtitle"`                                           |
| `desc`          | `string`    | Description text shown in the card                   | `"This is a brief course description..."`                    |
| `tags`          | `string[]`  | Array of tags to display in the card                 | `["Python", "TensorFlow", "Pandas"]`                          |
| `startDate`     | `string`    | Start date of the course                             | `"Starts April 12, 2025"`                                     |
| `duration`      | `string`    | Duration of the course                              | `"6 months"`                                                  |
| `price`         | `number`    | Current price of the course                         | `6999`                                                        |
| `originalPrice` | `number`    | Original price of the course                        | `8999`                                                        |
| `discount`      | `string`    | Discount text to display                             | `"Save 22%"`                                                  |
| `img`           | `string`    | URL of the image to display in the card             | `"https://images.pexels.com/photos/1181675/pexels-photo-1181675.jpeg?auto=compress&cs=tinysrgb&w=600"` |
| `link`          | `string`    | URL to redirect when the "Learn More" button is clicked | `"#"`                                                        |
| `ChaiTheme`     | `boolean`   | Whether to apply Chai-themed amber color styles      | `false`                                                      |
```
